## 前置知识

- GWAS部分

  - 教程：[GWA_tutorial](https://github.com/MareesAT/GWA_tutorial)
  - 基础原理和定义：[easyGWAS]([easyGWAS - Running GWAS easily over the web](https://easygwas.biochem.mpg.de/))
  - 基础定义
    - GWAS:
    - SNP:在基因组DNA序列中**单个核苷酸**（A、T、C、G）发生替换、插入或缺失的遗传变异
    - indels:基因组序列中**单个或多个连续核苷酸的插入或缺失**的遗传变异
    - P值（P-value）：用于衡量关联性显著性的统计指标
    - Bonferroni校正：多重检验校正方法，显著性阈值/检验次数。
    - FDR（False Discovery Rate）：假发现率，在所有显著结果中假阳性的比例
    - 连锁不平衡（Linkage Disequilibrium, LD）：相邻SNP位点间非随机关联的现象
    - LD衰减（LD Decay）：连锁不平衡随物理距离增加而减弱的现象。
    - 等位基因（Allele）：同一基因座位上不同形式的基因。
    - 基因型（Genotype）：个体在特定基因座位上的等位基因组合。
    - 表型（Phenotype）：个体可观察到的性状或疾病状态。
    - 效应量（Effect Size）：遗传变异对表型影响的大小，常用OR值或β系数表示。
    - 队列研究（Cohort Study）：追踪人群随时间发展，观察基因型与疾病发生关系的研究设计。
    - 数量性状（Quantitative Trait）：连续分布的性状（如身高、体重、血压等）。
    - 数量性状位点（Quantitative Trait Locus, QTL）：影响数量性状的基因组区域。
    - eQTL（expression QTL）：影响基因表达水平的遗传变异位点。
    - 曼哈顿图（Manhattan Plot）：GWAS结果的可视化图形，显示各染色体上SNP的显著性水平。
    - QQ图（Quantile-Quantile Plot）：检验观察到的P值分布与期望分布一致性的图形。
    - 群体分层（Population Stratification）：研究人群中存在不同祖先群体导致的假阳性关联。
    - 协变量（Covariate）：在分析中需要调整的变量（如年龄、性别、BMI等）。
    - 遗传力（Heritability）：性状变异中由遗传因素解释的比例。
    - 复杂数量性状（Complex Trait）：受多个基因和环境因素共同影响的性状。
